======================================自启动====================================

有两种方法，1. 手动配置启动方式；2. 设置成系统服务(可以查看chkconfig 注册系统服务)

手动配置是最简单的方式：（脚本名字dwj.sh）

1.在/etc/rc.d/init.d/下添加相应的启动脚本

2.修改启动脚本的权限 <br>
>[root@dwj ~]# chmod 755 dwj.sh

3.修改/etc/rc.local文件，最后一行添加第一步的脚本绝对路径

例子：dwj.sh脚本内容
```
#! /bin/bash
cd /opt
/opt/SimaVideoAccess/VAS_Access
```
注：VAS_Access 是程序名
